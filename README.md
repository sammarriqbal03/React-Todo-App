# React-Todo-App
# 📝 React Todo App

A modern Todo application built with React (Vite) featuring task management, filters, priorities, search, and dark/light theme support.
Features
- ➕ Add tasks
- ✏️ Edit tasks
- ❌ Delete tasks
- ✅ Mark complete/incomplete
- 🔍 Search tasks
- 📂 Filter (All / Active / Completed)
- ⚡ Priority system (High / Medium / Low)
- 🌙 Dark / Light mode
- 💾 Local storage support (data saved automatically)
- 📊 Live task stats
 🛠 Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS3 (Glassmorphism + Modern UI)
- LocalStorage API
